# StayOn v0.1.4 — early beta

**Recommended build for third-party testers.** Includes all fixes from v0.1.2–v0.1.3 plus CPX reward accuracy and polish.

Install: `extension/stayon-0.1.4.vsix` → reload → **StayOn: Set Up**

**Host:** redeploy `web/` and set `CPX_CURRENCY_FACTOR=700` (match CPX dashboard).

---

## Highlights

| Area | Fix |
|------|-----|
| **Windows Agent panel** | Inbox file fallback when hook HTTP blocked (v0.1.2) |
| **Setup self-test** | No longer leaves panel on “Cursor is working” (v0.1.3) |
| **CPX rewards** | Credit `amount_local` from postback — matches CPX survey Points (1000 = $1) |
| **Cash display** | Survey earnings only (Learn/Perks excluded); `≈ $X.XX` from server balance |
| **Windows alert** | Chime + attempt to focus Cursor when agent finishes |
| **Privacy** | `/privacy` page for beta (survey profile + hooks) |

---

## Tester checklist

1. Install VSIX → reload Cursor  
2. Open project folder → **StayOn: Set Up**  
3. Submit **Agent** prompt → panel opens  
4. **Open in browser** → complete survey  
5. Points sync ~30s; **Wallet → View earnings online**  
6. **No cash withdrawal yet** — claim payout is a later release  

---

## Known limitations

- Learn/Perks points increase total balance but not survey earnings  
- Pre-0.1.4 ledger rows may have incorrect point amounts (new surveys use correct math)  
- Windows focus-from-browser may not work on every OS focus policy  
- Open VSX listing not required for private beta (VSIX sideload)
