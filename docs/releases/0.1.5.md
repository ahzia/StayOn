## Publish Open VSX (manual)

You need `OVSX_PAT` from [open-vsx.org](https://open-vsx.org). Full steps: [docs/23_open_vsx_publish.md](../23_open_vsx_publish.md).

```bash
export OVSX_PAT="your-token"
cd extension
npx ovsx create-namespace stayon -p "$OVSX_PAT"   # once
npm run publish:ovsx
```
